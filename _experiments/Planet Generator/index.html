--- layout: demo title: Planet Generator disc: Generate random planets using procedural noise. tags: GPU WebGL langs: GLSL JavaScript info: - "This project is a simple demo of what can be created using my library [gl-Noise](https://github.com/FarazzShaikh/glNoise). It uses Farctal Noise made by layering multiple layers of Simplex Noise together. The reuslt is masked by an aditional layer of perlin noise to give the continents a smooth falloff into the ocean. The water is made by setting the height of the sphere to a constant. The terrain is generated in a VErtex Shader.

The colors are generated in the Fragment Shader where each pixel gets a color depending on its vertical position. This verticle position is passed in from the Vertex Shader.
